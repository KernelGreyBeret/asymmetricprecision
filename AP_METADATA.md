# AP Metadata

**Version:** 0.1  
**Purpose:** Thinking metadata for essays, frameworks, books, laboratories, and Atlas nodes.

---

## Why Metadata Exists

AP metadata is not just SEO.

It helps people understand what they are about to read, where it fits, what it connects to, and how to continue learning.

Metadata should make the system visible.

## Difficulty Scale

1. Entry point
2. Accessible systems idea
3. Practitioner-level concept
4. Advanced synthesis
5. Dense or foundational reference

---

## Reflection Metadata

Each major content item should eventually include at least one reflection prompt.

Examples:

- What system did this help you see more clearly?
- What assumption did this challenge?
- Where have you observed this pattern?
- What would change if your organization understood this idea?
