# AP Accessibility Hardening

## Principle

**Access is not an accommodation. Access is part of understanding.**

AP exists to help people understand complex systems. If a visitor cannot navigate the site with a keyboard, screen reader, touch interface, or reduced-motion preference, the system has failed its own mission.

## Sprint 14 Adds

- Skip link support
- Stronger focus-visible states
- Active nav states with `aria-current`
- Reduced motion behavior
- Forced-colors support
- Live-region conventions for hydrated AP surfaces
- External-link state labeling

## The AP Test

Ask one question:

> Does this interaction make understanding available to more people?

If yes, it belongs.
