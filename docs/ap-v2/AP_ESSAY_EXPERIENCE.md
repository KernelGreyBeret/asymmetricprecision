# AP Essay Experience

Version 0.9

## Sprint Principle

**An essay is not a post. An essay is a lens.**

An AP essay should not be treated as a blog entry, update, opinion piece, or chronological artifact. It should help a reader see a difficult system more clearly.

## Essay Requirements

Every AP essay should answer four questions:

1. What system does this reveal?
2. What hidden force, pattern, or relationship does it name?
3. What mental model does it improve?
4. Where does the reader go next?

## Experience Pattern

Each essay should include:

- A lens statement
- A central question
- Metadata surface
- Domain and difficulty indicators
- Atlas connections
- Continue-the-thought surface
- Learning checkpoint

## Success Measure

The essay succeeds when the reader leaves with a better lens than they arrived with.

Traffic is secondary.
Engagement is secondary.
Understanding is primary.
