# AP Atlas Workspace

## Sprint 33 — Atlas Workspace

**The map should have room to become a workspace.**

Sprint 33 changes the Atlas from a narrow three-column projection into a wider map-first workspace.

## What changed

- The Atlas map now occupies the full width above the node list and detail surfaces.
- A map workspace mode lets the learner expand the Atlas into a near-full-screen environment.
- Move Nodes mode lets the learner drag nodes into a clearer arrangement for their own understanding.
- The current map projection can be opened as an SVG image.
- The current map projection can be downloaded as an SVG blueprint.

## Why it matters

The Atlas is not a sidebar. It is the spatial expression of AP's relationship model.

If the map is too narrow, the system becomes visually compressed. A compressed map creates uncertainty instead of reducing it.

Sprint 33 gives the Atlas room to breathe while preserving the bounded Stability Core from Sprint 32.
