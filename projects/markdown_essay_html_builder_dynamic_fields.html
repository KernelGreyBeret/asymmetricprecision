Markdown Essay HTML Builder

Offline browser tool: Markdown essay + HTML template in, finished essay HTML file out.

1. Essay Metadata

Slug is generated automatically from the title: lowercase, punctuation removed, spaces become dashes.

Custom Template Fields

Example: placeholder {{AUTHOR}} with value Tommy Burke. You may type AUTHOR or {{AUTHOR}}; the tool normalizes it.

2. Markdown Essay Body

Also supports normal Markdown blockquotes using lines that start with >.

Supported Markdown

Headings: #, ##, ###
Bold: **text** or __text__
Emphasis: *text* or _text_
Lists: lines starting with -, *, or +
Paragraphs, blank-line breaks, manual line breaks, and blockquotes.

3. HTML Template

Default placeholders: {{TITLE}}, {{TITLE_SLUG}}, {{SUBTITLE}}, {{ESSAY_NUMBER}}, {{READING_TIME}}, {{ESSAY_BODY}}, plus any custom placeholders you add.
