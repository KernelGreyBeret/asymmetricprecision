# AP v2 Sprint 20 — Atlas Cartography

## Principle

**A map should reveal structure, not merely prove links exist.**

## Install

Drop this package into the root of the AP preview repo and overwrite existing files.

## Adds / Updates

- `assets/system/ap-atlas.js`
- `assets/system/ap-atlas-cartography.css`
- `assets/system/ap-system.css`
- `atlas/index.html`
- `docs/ap-v2/AP_ATLAS_CARTOGRAPHY.md`

## Notes

This sprint makes the Atlas more map-like through relationship density, curved edges, domain hulls, labels, metrics, and a legend. It does not replace the need for future page-level mini maps or inline contextual surfaces. It prepares the Atlas for them.
