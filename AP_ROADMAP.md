- Manifesto
- Oath
- Vocabulary
- Taxonomy
- Metadata model
- Design system philosophy
- Roadmap
- Homepage v2
- About v2
- Contact v2
- Philosophy page
- Start Here page

## Sprint 2 — Site Architecture

Goal: make the repository and navigation reflect the philosophy.

- Introduce Philosophy page
- Introduce Start Here page
- Rename Projects to Laboratories
- Update global navigation
- Add AP footer language
- Add content connections pattern
- Add reflection pattern

## Sprint 3 — Content System

Goal: make essays, frameworks, books, field notes, and laboratories part of one connected learning system.

- Add metadata to content
- Add domains
- Add principles
- Add related connections
- Add reflection questions
- Add learning paths

---

## Sprint 4 — Atlas Foundation

Goal: create the first static Atlas.

- Define node data
- Define edge data
- Build static Atlas page
- Link content to Atlas
- Add domain clusters

---

## Sprint 5 — Hidden Systems Visual Language

Goal: make AP visually recognizable through meaningful systems-oriented design.

- System map components
- Connection cards
- Evidence blocks
- Telemetry labels
- Diagram patterns
- Atlas visual language

---

## Sprint 6 — Interactive Understanding

Goal: add interaction only where it increases learning.

- Reflection checkpoints
- Observation submissions
- Learning journey signup
- Lightweight feedback capture
- No vanity metrics

---

## Launch — AP v2.0

Goal: merge the complete experience into main.

Success criteria:

> A visitor leaves understanding the system better than when they arrived.
