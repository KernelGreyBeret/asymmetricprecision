# Sprint 32 — Atlas Stability Core

**Principle:** A map must remain usable before it becomes elaborate.

## Install

Drop these files into the repo root and overwrite existing files:

- `assets/system/ap-atlas.js`
- `assets/system/ap-system.js`
- `assets/system/ap-system.css`

Add:

- `assets/system/ap-atlas-stability.css`
- `docs/ap-v2/AP_ATLAS_STABILITY_CORE.md`

## Notes

This sprint intentionally makes `ap-atlas.js` the only Atlas renderer. Later Atlas visual sophistication should be reintroduced into this stable core, not layered on top as competing renderers.
