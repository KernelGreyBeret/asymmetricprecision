# AP Patterns

Reusable HTML patterns for Asymmetric Precision.

These are not generic components. Each pattern expresses part of the AP system:

- `ap-oath.html` introduces the mindset before content.
- `learning-checkpoint.html` measures understanding instead of engagement.
- `connection-panel.html` treats relationships as first-class citizens.
- `atlas-node.html` seeds the future Atlas model.
