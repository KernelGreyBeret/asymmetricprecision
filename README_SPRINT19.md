# Sprint 19 — Atlas Blueprint Engine

Principle: **A map is not navigation. A map is a model of relationship.**

This sprint upgrades the Atlas from a filterable relationship surface into a true blueprint-style projection engine.

## Adds

- Atlas projection control
- System Map projection
- Focused Neighborhood projection
- Domain Cluster projection
- Journey Path projection
- Active relationship labels in the map
- Domain cluster labels
- Framework card boundary tightening
- Atlas blueprint documentation

## AP Standard

The Atlas should not be a list of AP resources.

It should be a way to see the AP system.
